## 基本指令

### 顯示目前所在目錄

```bash
pwd
```
### 列出目錄內容

```bash
ls
```
### 切換目錄

```bash
cd 目錄名稱
cd ..
cd ~
```
### 建立資料夾

```bash
mkdir 資料夾名稱
```
### 建立檔案

```bash
touch 檔案名稱
```
### 查看檔案內容
```bash
cat 檔案名稱
```
### 刪除檔案 / 資料夾（小心）

```bash
rm 檔案名稱
rm -r 資料夾名稱
```

## 常用快捷鍵

| 快捷鍵      | 功能    |
| -------- | ----- |
| Tab      | 自動補齊  |
| Ctrl + C | 中斷指令  |
| Ctrl + L | 清空畫面  |
| ↑        | 上一個指令 |
| ↓        | 下一個指令 |

## 基本觀念
- 指令大小寫有分別，請注意所有打在指令欄的文字
- `~` 代表家目錄
